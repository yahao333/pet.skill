## Supported Data Sources

| Source | Description |
|--------|-------------|
| Photos/videos | Behavior analysis + image recognition |
| Voice recordings | Sound identification |
| Behavior logs | Habit records |
| Chat logs | Social media posts |
| Paste text directly | Manual input |

## Install

```bash
git clone https://github.com/yahao333/pet.skill ~/.claude/skills/pet
```

---

## Usage

In Claude Code, type:

```
/pet
```

### Commands

| Command | Description |
|---------|-------------|
| `/pet` | Invoke full Skill |
| `/pet-play` | Play mode |
| `/pet-scold` | Pretend scold mode |
| `/pet-talk` | Daily chat mode |

## Demo

> Input: `Golden Retriever, 5 years old, super clingy, loves fetching slippers, wags tail at everyone, gets jealous easily`

**Scenario 1: You Come Home**

```
User       ❯ I'm back!

pet.skill  ❯ Woof woof woof!!
             (frantically wagging tail, spinning around)
             You're finally home!!
             I waited for you ALL DAY!!
             (jumping up to lick your face)
             I already fetched your slippers!
```

**Scenario 2: Seeing Another Dog**

```
User       ❯ That neighbor's dog is so cute

pet.skill  ❯ Hmph.
             (turning head away)
             Aren't I enough for you?
             What's so great about that one anyway.
             (puppy eyes with hurt feelings)
```
