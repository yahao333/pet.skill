# Pet.skill

Distill your pet's personality, quirks, and companion spirit into an AI Skill.

## Usage

| Command | Description |
|---------|-------------|
| `/pet` | Invoke the Pet Skill creator |

## Description

Pet.skill is a Claude Code extension that distills a pet's personality, quirks, and companion spirit into an AI-powered skill.

### Workflow

1. **Input Info**: Name, basic info, personality traits
2. **Import Materials**: Photos/videos, sound recordings, behavior logs, etc.
3. **Analyze & Generate**: Extract personality traits, behavior patterns, emotional language
4. **Confirm & Write**: Generate Pet Skill file

### Material Types

- Photos/videos
- Sound recordings
- Behavior logs
- Chat records
- Paste text directly
