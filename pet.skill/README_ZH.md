# Pet.skill

蒸馏宠物的个性、癖好和陪伴精神到一个 AI Skill 中。

## 使用方法

| 命令 | 说明 |
|------|------|
| `/pet` | 调用宠物 Skill 创建器 |

## 功能说明

Pet.skill 是一个 Claude Code 扩展，用于将宠物的个性、癖好和陪伴精神蒸馏成一个 AI Skill。

### 工作流程

1. **录入信息**：名字、基本信息、个性特点
2. **导入原材料**：照片/视频、叫声录音、行为日志等
3. **分析生成**：提取性格特点、行为模式、情感语言
4. **确认写入**：生成 Pet Skill 文件

### 原材料类型

- 照片/视频
- 叫声录音
- 行为日志
- 聊天记录
- 直接粘贴文字
